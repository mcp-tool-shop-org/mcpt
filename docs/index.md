# mcpt

CLI for discovering, installing, and running MCP Tool Shop tools.

## Key Features

- **Official registry client** for the mcp-tool-registry
- **Workspace management** via mcp.yaml — declare, pin, and share tool sets
- **Trust tiers** (trusted / verified / neutral / experimental) with risk auras
- **Safe-by-default execution** — tools run in stub mode unless promoted
- **Registry pinning** for reproducible builds
- **Rich TUI** with --plain and NO_COLOR support

## Install / Quick Start

```bash
pip install mcp-select
# or
npx @mcptoolshop/mcpt
```
